Atributos HTML comunes: id, class, href, src, alt

El atributo id se usa para identificar un elemento de forma única.

El atributo class se usa para agrupar elementos con estilos comunes.

El atributo href se usa en enlaces para especificar la URL de destino.

El atributo src se usa en imágenes para especificar la URL de la imagen.

El atributo alt se usa en imágenes para proporcionar un texto alternativo.
