# Inputs html

Este es un proyecto de bun, para iniciarlo primero, he editado el gitignore para ignorar las carpetas node_modules.

Antes de todo se debe editar el gitignore desde la carpeta raiz del proyecto:

**/*node_modules**

Con esto se ignoran todas las carpetas node_modules que se encuentren en cualquier subcarpeta del proyecto.

Luego se ha creado un directorio dentro he hecho:

```bash
bun init
```

